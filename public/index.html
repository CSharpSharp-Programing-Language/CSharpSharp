Say hello to C##

A tiny, friendly language that blends C#-style syntax with C-like simplicity. Great for learning, tinkering, and quick scripts.

Download Get Started
MIT Licensed • Cross‑platform via Python 3 • Single‑file interpreter
examples/hello.c## 
using System;

int Main() {
  Console.WriteLine("Hello from C##!");
  return 0;
}

Features

Compact, batteries‑included core so you can focus on ideas.

Simple Syntax

C‑style blocks, `if/else`, `for`, `while`, and C#‑like `foreach`.

Lists Built‑in

`List()` with `push_back`, indexing `xs[i]`, and `size()`.

Console I/O

`Console.WriteLine(...)` and `Console.ReadLine()` for quick scripts.

Single‑file Interpreter

Run with Python 3.9+: `python3 src/cshs.py run examples/hello.c##`

REPL

`python3 src/cshs.py repl` for a quick interactive loop.

MIT Licensed

Use it, tweak it, ship demos freely.

Examples

Loops & Lists

int Main() {
  var xs = List();
  for (var i = 0; i < 5; i = i + 1) {
    xs.push_back(i);
  }
  foreach (var n in xs) { Console.WriteLine(n); }
  return 0;
}

Functions

int Square(int x) { return x * x; }

int Main() {
  Console.WriteLine(Square(7));
  return 0;
}

Docs (v0.2)

Basics

  • Statements end with ;
  • Variables: var x = 10; or int x = 10;
  • Control flow: if, else, while, for, foreach
  • Console: Console.WriteLine(...)

Lists

var xs = List();
xs.push_back(42);
Console.WriteLine(xs[0]);
Console.WriteLine(xs.size());
